An alias is a **shortcut for a command or set of commands**.

Example:

```bash

```

Now, typing `ll` runs `ls -la`.

## ğŸ”§ Create Temporary Alias

## ğŸ”§ Make Aliases Permanent

# ğŸ¨ 4. Shell Prompt Customization (PS1)

## âœ… What is PS1?

`PS1` defines how your **command prompt** looks.

Example default:

```

```

## ğŸ”§ Change Prompt Temporarily

```bash

```

â¡ï¸ Now your prompt looks like:

```

```

## ğŸ”§ Set Permanent Custom Prompt

1. Open `.bashrc`:

   ```bash
   nano ~/.bashrc
   ```

2. Add your new prompt:

   ```bash
   export PS1="\u@\h:\w\$ "
   ```

   * `\u` = username
   * `\h` = hostname
   * `\w` = current directory

3. Reload:

   ```bash
   source ~/.bashrc
   ```
