## âœ… What is a Package?

A **package** is a compressed bundle of software files. It contains:

* Program files
* Configuration files
* Metadata for installation

Linux uses **package managers** to install, update, and remove these packages.

---

## ğŸ” What is a Package Manager?

A **Package Manager** is a tool that helps:

* Download software
* Install and configure it
* Resolve dependencies (other required software)
* Update and remove software

## 1ï¸âƒ£ APT (Advanced Package Tool)

### ğŸ”¹ What is apt?

APT is the main package manager on Ubuntu. It downloads and installs packages from Ubuntu's repositories.

### ğŸ”¹ Common apt Commands

| Action                        | Command Example            |
| ----------------------------- | -------------------------- |
| Update package lists          | `sudo apt update`          |
| Upgrade installed packages    | `sudo apt upgrade`         |
| Install a new package         | `sudo apt install firefox` |
| Remove a package              | `sudo apt remove firefox`  |
| Search for available packages | `apt search vlc`           |
| Show package details          | `apt show vlc`             |
| List installed packages       | `apt list --installed`     |
| Remove unused packages        | `sudo apt autoremove`      |
| Clear download cache          | `sudo apt clean`           |

### ğŸ” Example: Installing VLC

```bash

```

### ğŸ”¹ Common dpkg Commands

### ğŸ” Example: Installing Chrome

# âœ… Final Tips for Ubuntu Users

* Prefer **apt** for most software.
* Use **snap** for universal apps like Spotify, VSCode.
* Use **dpkg** when you download `.deb` files.
* Run `sudo apt update && sudo apt upgrade` weekly.
* Clean up with `sudo apt autoremove` and `sudo apt clean` regularly.
